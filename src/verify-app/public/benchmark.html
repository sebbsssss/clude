06 — Differentiators

Key Differentiators

Capabilities that no competitor — commercial or academic — currently combines.

01

The Only Memory System That Dreams

5-phase dream cycle: consolidation, compaction, reflection, contradiction resolution, emergence. No competitor implements autonomous offline processing.

02

Active Contradiction Resolution

Proactively detects conflicting memories via database RPC, resolves them through LLM mediation, creates resolves links, and accelerates decay on the weaker memory.

03

Biologically-Inspired Decay

4 type-specific exponential decay rates: episodic 0.93, procedural 0.97, semantic 0.98, self_model 0.99. Events fade. Knowledge persists. Identity endures.

04

3 Vectors Per Memory

Per-fragment embeddings decompose each memory into summary, content_chunk, and tag_context vectors. Competitors embed once. Clude matches at three semantic levels.

05

7-Phase Recall Pipeline

Vector search, metadata filtering, candidate merging, composite scoring, entity-aware expansion, bond-typed graph traversal, type diversity injection.

06

On-Chain Memory Commitment

Every memory gets SHA-256 hashed and written to Solana. Verifiable, immutable, tamper-evident. No other memory system has blockchain integration.
